# iOS App

SwiftUI iOS-first client for Anagram Master.

## Minimum Target

- iOS 17.0

## Generate Project (macOS)

1. Install Xcode 15+
2. Install XcodeGen: `brew install xcodegen`
3. In `ios-app/AnagramArena`, run: `xcodegen generate`
4. Open `AnagramArena.xcodeproj`

## Phase 0 scope

- App bootstrap with home screen placeholder.
- Project spec and test target scaffold.
- Gameplay flow and engine integration start in Phase 1.
